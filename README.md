# Backend Flask para login con MySQL

Este proyecto tendrá:
- Un backend en Python usando Flask.
- Conexión a MySQL para validar el usuario administrador.
- Almacenar los datos de login de usuarios normales.
- El admin podrá ver los datos ingresados por los usuarios en una interfaz especial.

## Estructura inicial

- app.py (servidor Flask)
- requirements.txt (dependencias)
- templates/
  - admin.html (interfaz admin)
  - login.html (interfaz login)

## Siguiente paso
Se creará el archivo app.py y requirements.txt, y se configurará la conexión a MySQL.
